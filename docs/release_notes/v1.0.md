# ClinOCR-Bench v1.0

First public release of **ClinOCR-Bench** — a benchmark for clinical OCR.

## What's included

`ClinOCR-Bench-v1.0.zip` (~114 MB) contains the full benchmark as raw files — no account or special tooling required, just download and unzip:

- **`scans/`** — document images, grouped by artifact subset
- **`ground_truth/`** — clean, human-audited transcriptions for every document
- **`oneshot_lookup.csv`** / **`oneshot_lookup.json`** — predefined one-shot demonstrations for reproducible 0-shot / 1-shot evaluation

## Dataset at a glance

- **384 documents** across **6 subsets** (64 each) and **16 templates**
- Subsets isolate distinct real-world artifacts: `Normal-quality`, `Handwriting`, `Poor-quality`, `Rotation`, `Tables`, plus a `Mixed` subset combining multiple artifacts
- **Train / test split**: 56 held-out exemplars + 328 scored eval documents
- Supports three regimes: **0-shot**, **1-shot (homogeneous)**, and **1-shot (heterogeneous)**
